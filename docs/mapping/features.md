# 기능 매핑 (구버전 우위 → 신버전 적용 대상)

| --- | --- | --- | --- | --- |
| 뱃지/인증 | certifications, 전문가 apply, trust/expert 뱃지 표시 | verification만 존재, 뱃지 없음 | P0 | 인증 완료/전문가 뱃지 표시, 관리 패널 추가 |
| 온보딩 개인화 | 비자/관심사 수집, 피드 반영 | 온보딩 화면만 있음, 개인화 미약 | P0 | 비자/관심사 필드 저장, 추천 연동 |
| 토픽/구독 | topics, topic_subscriptions, 토픽 피드 | 카테고리만, 구독 없음 | P1 | 토픽 슬러그 라우트, 구독 API, 구독 피드 탭 |
| 실데이터/SEO | 실데이터 SSR, 메타 일치 | 일부 목업, 메타/JSON-LD 불일치 | P0 | mock 제거, `generateMetadata`/JSON-LD 정합화, sitemap 보강 |
| 소셜 그래프 | 추천 사용자, 신뢰 지표, 팔로우 기반 피드 | 팔로우만 기본 제공 | P1 | trust_score/helpful_answers 지표, 추천 패널, 팔로우 피드 강화 |
| 랭킹 | user-rank 페이지/패널 | 없음 | P2 | 주간/월간 랭킹, 지표 기반 정렬 |
| 미션/레벨 | missions, 경험치/레벨 | 없음 | P2 | 일일/주간 미션, 레벨 배지(지갑 제외) |
| FAQ/헬프 | FAQ 카테고리/아이템, 관리자 | 없음 | P3 | `/faq`, `/admin/faq` |
| 배너/이벤트 | site_banners, 이벤트 페이지 | 없음 | P3 | 홈 배너, 이벤트 상세, 관리자 |
| 비자 매칭 | 전공-직무-비자 요건 매칭 흐름 | 일반 Q&A 중심 | P1 | 질문 템플릿/체크리스트/매칭 위저드 |
| 콘텐츠 품질 | 신고/모더레이션 도구 강화 | 기본 신고 수준 | P1 | content_reports 스키마, 운영자 뷰, 레이트리밋 |
| 마케팅 연계 | 카드뉴스/숏폼 임베드, 공유 최적화 | 기본 OG 수준 | P2 | OG/Twitter/Zalo 프리뷰, UTM 버튼, 임베드 섹션 |
