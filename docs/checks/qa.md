# QA / 품질 게이트 체크

## 기본 게이트
- `npm run lint`, `npm run build` 필수 통과.
- 다국어 스모크: ko/en/vi 홈/로그인/검색/상세 라우트 SSR 응답 및 메타 확인.
- 목업 데이터 검출: mockPosts 등 하드코딩 데이터 제거 여부 점검.

## 기능 스모크 (우선)
- 뱃지/인증: 인증 완료 시 UI 표시, 전문가 뱃지 노출, 인증 실패/오류 핸들링.
- 온보딩: 비자 상태/관심사 저장, 홈 추천 반영 여부.
- 토픽/구독: 구독/취소, 구독 피드 탭, 토픽 상세 SSR.
- 소셜 그래프: 팔로우/언팔로우, 추천 패널 로드, trust 지표 표시.
- 랭킹/미션: XP 누적, 미션 완료 처리, 랭킹 정렬.
- FAQ/배너/이벤트: 리스트/상세 로드, 관리자 수정 후 반영.
- 비자 매칭: 템플릿/체크리스트 제출, 매칭 결과 표시.

## 회귀/에러 모니터
- API 4xx/5xx 로그 수집, 레이트리밋/스팸 대응 경고.
- 성능: LCP/INP/CLS 샘플 측정, 이미지/폰트 최적화 확인.
