# 마케팅 / OG / 공유 체크

## 공유/OG
- OG/Twitter 카드: title/description/thumbnail 언어별 생성, 베트남어·한글 병기 허용.
- Zalo 미리보기: Open Graph 메타 활용, 썸네일 1200x630, 로고 안전 영역 확보.
- 카드뉴스/숏폼: 페이지 내 임베드 섹션과 공유 버튼(페북/TikTok/YouTube/Zalo) 배치.
- UTM 템플릿: `utm_source={platform}&utm_medium=social&utm_campaign={campaign}&utm_content={content_id}` 공통 적용.

## 채널별 액션
- Facebook: OG 테스트, 페북 공유 버튼/딥링크, 그룹/페이지용 썸네일 사전 검증.
- Zalo: OA 링크 공유 시 프리뷰 확인, QR 코드 제공, 메시지 브로드캐스트 시 단축 URL+UTM.
- TikTok/YouTube Shorts: 랜딩 링크/QR 삽입, 클릭 전환 낮으므로 인지도 트랙으로 분리.

## 데이터/트래킹
- 리퍼러/UTM 수집 → 가입/질문/답변 전환 퍼널 대시보드 설계.
- 인기 콘텐츠/캠페인별 전환율 집계: 카드뉴스 저장/공유, 숏폼 클릭아웃.

## UX 배치 제안
- 홈/포스트 상세 상단: 공유 버튼(페북/Zalo), 하단: 카드뉴스/숏폼 임베드.
- 이벤트/캠페인 페이지: OG 썸네일, CTA 버튼에 UTM 기본값 포함.
